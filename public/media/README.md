# Public Media Structure

This is the web-ready asset library for Bali YTTC.

Use paths from this folder directly in the app, for example:

```ts
const hero = "/media/01-home/hero/bali-yttc-home-hero-bg.png";
const accommodation = "/media/03-accommodation/rooms/private-villa-room-01.jpg";
const teacher = "/media/05-instructors/vivek-kalura-lead-teacher.jpg";
```

For a complete file list, see `public/media/media-inventory.json`.

## Section Map

- `00-brand`: logos, badges, certification marks
- `01-home`: homepage hero and homepage video
- `02-courses`: course-specific media by duration
- `03-accommodation`: campus, rooms, pool, facilities
- `04-activities`: ceremony, practice, workshops, beach yoga
- `05-instructors`: teacher profile images
- `06-testimonials`: testimonial videos and future review images
- `07-gallery`: public gallery categories
- `08-documents`: brochures, manuals, certificate PDFs
